# Getting Support

Thank you for using SocialConnector! Here are the best ways to get help.

## Read the Docs First

- [README](README.md) — Overview, installation, and quick start
- [CONTRIBUTING](CONTRIBUTING.md) — How to set up the project locally
- [docs/](docs/) — Architecture, provider guides, and API reference

## Bug Reports

If you've found a bug, please [open a GitHub Issue](https://github.com/naveenairani/socialconnect/issues/new?template=bug_report.md).

Before opening a new issue, please:
1. Search [existing issues](https://github.com/naveenairani/socialconnect/issues) to avoid duplicates
2. Include your Python version, OS, and `socialconnector` version
3. Provide a minimal reproducible example

## Feature Requests

Have an idea? [Open a feature request](https://github.com/naveenairani/socialconnect/issues/new?template=feature_request.md). We love community input!

## Questions & Discussions

For general questions, usage help, or ideas:
- Use [GitHub Discussions](https://github.com/naveenairani/socialconnect/discussions)

## Security Issues

Please **do not** open public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
