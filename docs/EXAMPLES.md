# Examples

`examples/` — runnable scripts showing common usage patterns.

## Files

- **basic_send.py** — send a message with one provider
- **multi_provider.py** — send same message to Telegram + Slack + Discord
- **webhook_server.py** — start webhook server, handle incoming messages
- **custom_provider.py** — implement and register a custom provider
- **media_upload.py** — send images/files across platforms

Each example should be <50 lines and self-contained.
